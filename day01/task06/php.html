PHP: Hypertext Preprocessor[5], plus connu sous son sigle PHP (sigle auto-référentiel), est un langage de programmation libre[6], développé le 8 juin 1995 principalement utilisé pour produire des pages Web dynamiques via un serveur web[5], mais pouvant également fonctionner comme n'importe quel langage interprété de façon locale. PHP est un langage impératif orienté objet.
